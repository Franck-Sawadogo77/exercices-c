## Structure des Exercices

### 01 - Variables et Calculs
Manipuler les variables, les types de données et effectuer des calculs mathématiques simples.

### 02 - Conditions
Utiliser les structures conditionnelles (if, else, switch) pour prendre des décisions.

### 03 - Boucles
Implémenter des boucles (for, while, do-while) pour répéter des actions.

### 04 - Fonctions
Créer et utiliser des fonctions pour modulariser le code.

### 05 - Tableaux 1D
Travailler avec les tableaux à une dimension.

- notes_dessus_dessous_moyenne.c
- tableau_copie_dans_tableau.c
- tableau_max_et_min.c
- tableau_nombres_paires.c
- tableau_plus_petit_au_plus_grand.c
- tableau_somme_et_moyenne.c
- tableau_somme_moyenne0.c

### 06 - Tableaux 2D
Manipuler les tableaux à deux dimensions (matrices).

- tableau2d_moyenne_plusforte_plusfaible_moyenne.c
- tableau2d_recherche.c
- tableau_2D_taille_variable.c
- tableau_2d.c
- tableau_2d_diagonale.c
- tableau_2d_somme_ligne.c
- tableau_2d_transpose.c

### 07 - Chaines de Caracteres
Gérer les chaînes de caractères et les opérations sur celles-ci.

- carte_identite.c
- deux_mots_identiques.c
- fiche_etudiant.c
- mot_a_l'envers.c
- mot_de_passe_nom_utilisateur_tentatives.c
- mot_de_passe_texte.c
- mot_de_passe_verification.c
- nom_prenom.c

### 08 - Recursivite
Comprendre et implémenter des fonctions récursives.

### 10 - Valeurs Globales
Utiliser les variables globales et comprendre leur portée.

### 11 - Pointeurs
Maitriser les pointeurs, les références et la gestion mémoire.
